# DATA-ANALYTICS-USING-HEALTHCARE-DATASET

An end-to-end healthcare data analytics project- from raw data ingestion and cleaning through exploratory data analysis to interactive dashboard development.

The project uses the Healthcare Dataset by Prasad Patil sourced from Kaggle, a synthetically generated dataset simulating hospital patient records across multiple medical conditions, insurance providers, and admission types.


⚠️ Dataset Note: This is a synthetically generated dataset. All variables are artificially balanced, meaning no strong correlations exist between variables.
